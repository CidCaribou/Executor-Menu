## ‚öôÔ∏è Setup & Usage

<details>
<summary>üíª Option 1: Bookmarklet</summary>


1. Click on the `menu.js` file and copy the code  
2. Create a new bookmark and name it whatever you want  
3. add `javascript:` at the front then paste the code from `menu.js`  
4. Press **Save** ‚Üí Enjoy!
</details>

<details>
<summary>üìú Option 2: Userscript</summary>


1. Install a **Userscript Manager** (Tampermonkey, Violentmonkey, etc.)  
2. Open the `Userscript` Folder
3. Click on the code from `Menu.user.js` and click raw
4. Use the bookmarklet from `https://executor-official.netlify.app/install` to activate the userscript version  
4. Make sure **Developer Mode** is enabled in your extension settings
</details>

<details>
<summary>üõ∏ Option 3: Extension</summary>


1. Go the website and download the crx file  
2. Go to `chrome://extensions` and then turn on developer mode
3. Drag and drop the .crx file or load unpacked and select the .crx file 
4. Click the Executor icon when you are on a website to activate the script.
5. Only works on chromium browsers. (‚ö†Ô∏èDoes not work on Chrome)
</details>
